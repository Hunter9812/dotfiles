  相关模式：

  - 拆字反查（uU+拼音），拆字辅码（拼音+`+拆字辅码）
  - Unicode（U+Unicode 码位）
  - 数字、人民币大写（R+数字）
  - 农历（转写：N+八位数字；获取当前农历：全拼nl，双拼lunar）
  - 简易计算器（cC+算式）
  - UUID（uuid）
  - 特殊符号、字符输入（全拼v+首字母缩写；双拼V+首字母缩写）
